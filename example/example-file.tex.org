* Introduction

This is a simple org file, containing asciimath.

** Problem

Org does not have a direct method to export this file to PDF (because ascimath is there literally).

** Solution

I wrote a script ~convert-asciimath-in-org-to-tex.sh~ which converts the contained asciimath snippets to tex, saves this file with extension ~.tex.org~, then exports to tex and/or pdf. 

Then we can use regular org-export-to-pdf to convert the org file to PDF with latex.

** Formatting asciimath

*For the conversion and export to work, the asciimath text must be in backquotes.*

Take care of proper backquotes ending, although missing them does not cause any damage apart from incorrect export.
